# Citedesperles E-Commerce

This is a complete e-commerce platform for jewelry and accessories built with Next.js and Node.js.

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/lucdaniel402-wq/citedesperles-ecommerce.git
   ```
2. Navigate to the directory:
   ```bash
   cd citedesperles-ecommerce
   ```
3. Install dependencies for the server:
   ```bash
   cd backend
   npm install
   ```
4. Install dependencies for the frontend:
   ```bash
   cd ../frontend
   npm install
   ```
5. Create a `.env` file for environment variables.
6. Start the server:
   ```bash
   npm start
   ```
7. Start the frontend:
   ```bash
   npm run dev
   ```

## Features
- User authentication
- Shopping cart management
- Order tracking
- Product reviews
- Payment gateway integration
- Search and filter functionality
- Responsive design
